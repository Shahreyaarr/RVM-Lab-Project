## 📌 Project Overview

This project is developed as part of the Malware Analysis Lab to simulate and analyze different types of malware in a safe and controlled environment.

It provides an interactive dashboard where users can explore malware behavior without any real harm.

---

## 🎯 Objective

- Understand malware behavior  
- Simulate cyber attacks safely  
- Visualize payload execution  
- Improve cybersecurity skills  

---

## 🧪 Features

- 🦠 Virus Simulation  
- 🔐 Ransomware Simulation  
- 🎯 RAT Simulation  
- 📊 Interactive Dashboard  
- 📦 Payload Visualization
